snippet_colors ( void )
extern

This functions prints sample logs without colors and then with colors.

This prints sample logs by,

  • first disabling colors
  • then enabling colors
  • then color config is reverted to previous setting
    Note
    While streaming to a log-file colors wouldn't be printed

extern

This function prints a very long message hoping for overflow.

This function is currently attempting to print a log-message of length 200 which should overflow if LOG_MAX_DATA_LEN is below 200

Note
Additional data would be written for header (i.e., prefix of log-message)
