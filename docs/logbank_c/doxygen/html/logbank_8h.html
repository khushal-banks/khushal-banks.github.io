enum LOG_LEVELS

Available values for log-level in increasing priority order.

Enumerator
LOG_DEBUG 

recommended for sanity checks (intended for developers)

LOG_INFO 

recommended for general informations (code flow)

LOG_WARN 

recommended for warnings (abnormal values)

LOG_ERROR 

recommended for recoverable errors (can continue)

LOG_ISSUE 

recommended for non-recoverable errors (can continue with limited functionality)

LOG_FATAL 

recommended for blockers (system failure: can not continue)
