logbank_c

Software logging in C

This library aims to provide common logging functionality for large-scale C/CPP projects.

Features

  • Priority levels
  • Location parameters like, function name, file name and line number
  • Date/Time stamps
  • Retrieve recent logs
  • Parsing logs
  • Logging in files (by default logs are printed on stderr)

Checkout, Wiki for advanced features.

Motive

Software logging has a huge history and there are many sound alternatives available which can be used instead of this library. However, it was considered necessary to provide an in-house light-weight solution for our main project logbank - Management Tool for Debugging logs which resulted in this library.
