Software logging in C

This library aims to provide common logging functionality for large-\/scale C/\+CPP projects.\hypertarget{index_autotoc_md19}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md19}

\begin{DoxyItemize}
\item Priority levels
\item Location parameters like, function name, file name and line number
\item Date/\+Time stamps
\item Retrieve recent logs
\item Parsing logs
\item Logging in files (by default logs are printed on stderr)
\end{DoxyItemize}

Checkout, \href{https://github.com/khushal-banks/logbank_c/wiki}{\texttt{ Wiki}} for advanced features.\hypertarget{index_autotoc_md20}{}\doxysection{\texorpdfstring{Motive}{Motive}}\label{index_autotoc_md20}
Software logging has a huge history and there are many sound \href{https://github.com/topics/logging-library}{\texttt{ alternatives}} available which can be used instead of this library. However, it was considered necessary to provide an in-\/house light-\/weight solution for our main project \href{https://github.com/khushal-banks/logbank}{\texttt{ logbank}} -\/ Management Tool for Debugging logs which resulted in this library.

Consider motivating us by {\bfseries{putting a star to this repository}}.
